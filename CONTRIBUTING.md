# Contributing

Thanks for contributing to Smart Attendance System.

## Local Development

1. Install backend dependencies:
   - `cd backend`
   - `python -m pip install -r requirements.txt`
2. Install frontend dependencies:
   - `cd frontend`
   - `npm install`
3. Run backend and frontend in separate terminals.

## Pull Request Guidelines

- Keep changes focused and small.
- Do not commit generated files or secrets.
- Update documentation when behavior or setup changes.
- Verify the app still runs before opening a PR.

## Basic Verification

- Backend starts on `http://127.0.0.1:8000`
- Frontend starts on `https://127.0.0.1:5173`
- Core endpoints return successful responses
